Implements [[map-interface|Map]] interface.
Allow null keys and values.
Unsynchronized.
Constant-time performance for basic operations.
```java
HashMap<String, Integer> scores = new HashMap<String, Integer>();
scores.put("A", 31);
scores.put("B", 31);

scores.get("A");
```
